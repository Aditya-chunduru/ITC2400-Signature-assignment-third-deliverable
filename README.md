# Trophy Hunter's Domain - PlayStation Trophy Tracking Website

Welcome to Trophy Hunter's Domain, a responsive and user-friendly website dedicated to PlayStation trophy tracking, gaming roadmaps, and community engagement.

## Project Overview
This website allows users to browse featured PlayStation games, view detailed trophy roadmaps with difficulty ratings and estimated times, and connect with a vibrant community of trophy hunters. It includes pages for:

- Browsing games with trophy info
- Trophy roadmap links and details
- Community resources including Discord and forums
- Support services and validated contact form
- About and contact information pages

## Features
- Semantic HTML5 structure and client-side form validation
- Consistent, well-organized CSS styling with responsive design using flexbox and media queries
- Thorough navigation allowing access to all pages from anywhere in the site
- External links to PSNProfiles for official trophy guides and leaderboards
- Clear and professional color scheme for ease of use and aesthetics

## Technologies Used
- HTML5
- CSS3 (Flexbox, media queries)
- Client-side validation with HTML5 forms
